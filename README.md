## Function Descriptions

### `abs_val(x)`
Returns the absolute value of a number without using Python’s built‑in `abs()`.

### `lcm(*args)`
Computes the least common multiple of two or more integers using a custom GCD helper.

### `area_of_circle(r)`
Computes the area of a circle using π ≈ 3.141592653589793.
